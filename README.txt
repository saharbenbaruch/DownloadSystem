# downloadSystem
we implemented download system by using design pattern State for User' status - Begginer, Advanced, Proffesional.
we also used design Pattern Observer for notify Player when download reach 20% that it possiable to play and when reach 100% that User cant 
play anymore.
we used principle of use Interface in context of common interface for 3 kind of errors:
-downloadError, InternetError, noSpaceError.
